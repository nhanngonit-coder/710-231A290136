PHẦN 1: LÝ THUYẾT

Câu 1: position: absolute vs relative

Khác biệt chính:

  • relative: định vị so với chính nó, vẫn chiếm chỗ
  • absolute: định vị theo phần tử cha gần nhất có position, không chiếm chỗ

Ví dụ:

<div class="parent">
  <div class="box1">Relative</div>
  <div class="box2">Absolute</div>
</div>

<style>
.parent {
  position: relative;
}
.box1 {
  position: relative;
  top: 20px;
}
.box2 {
  position: absolute;
  top: 0;
  right: 0;
}
</style>

Ứng dụng:

  • relative: chỉnh vị trí nhẹ
  • absolute: popup, dropdown

Responsive:

  • Dùng sai absolute → vỡ layout mobile
  • Kết hợp flexbox, media query

Câu 3: let, const, var

So sánh:

  • var: function scope
  • let: block scope
  • const: block scope, không đổi

Ví dụ lỗi:

console.log(a); // undefined
var a = 5;

console.log(b); // lỗi
let b = 10;

Lỗi thường gặp:

for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}
// 3 3 3

Fix:

for (let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}

Debug:

  • F12 → Console
  • console.log()
  • Breakpoint

Câu 4: Event Delegation

Khái niệm:

Gắn event cho phần tử cha để xử lý cho phần tử con

Ví dụ:

<ul id="list">
  <li>Item 1</li>
</ul>

<script>
document.getElementById("list").addEventListener("click", function(e) {
  if (e.target.tagName === "LI") {
    alert(e.target.innerText);
  }
});
</script>

Lợi ích:

  • Tối ưu hiệu suất
  • Code gọn
  • Hỗ trợ phần tử động
